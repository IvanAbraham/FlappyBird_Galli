# FlappyBird_Galli

Juego creado por Lucas Galli y actualizado por Iván Abraham post versión 0.4.

Esta versión de FlappyBird es una recreación del juego original en lenguaje C++ con la biblioteca Raylib.
El jugador 1 tiene un solo control, "W" para saltar.
El jugaodor 2 al igual que el jugador uno cuenta con un solo control. Este salta utilizando la flecha superior.

Link al repositorio del proyecto original: https://github.com/Movious0207/FlappyBird_Galli
